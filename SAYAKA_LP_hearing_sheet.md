> これは制作物の種類ではなく**美意識の軸**なので、LPのビジュアルトーン・コピー・ターゲット設計すべてに通底させる。  
> → ターゲットは「装飾過多な広告チラシ的世界観の顧客」ではなく、「**抜け感・洗練・余白を欲している顧客**」になる。これは差別化の核。

## SECTION 3. 強み・差別化（モノクロ式：価値構造化）

ここがLPの“キー”です。一番時間をかけて書いてOK。

### 3-1. 顧客の声から逆算する
- 過去、顧客から「**こういうところが良かった**」と褒められた言葉 3つ：  
  1. **レスポンスが早い**  
  2. **デザインが綺麗**  
  3. **売上が上がった**  

- リピートしてくれる理由：  
  → **クオリティー**

### 3-2. 同業との違い
- 他のグラフィックデザイナーと比べて自分が明確に違うこと：  
  → **コミュニケーション能力**（デザインスキルではなく、結果的にコミュ力で選ばれている）  
- 「ここは他の人の方が上」と思うこと：  
  → 上には上がいる。とはいえ、**その中で常に努力し頑張っている**スタンス。（＝比較で勝負しない、自分を磨くスタンス）

### 3-3. 制作プロセスの哲学
- ヒアリング〜納品まで特に大事にしていること：  
  → **印刷物は印刷工程中に間違いが発見されるのが最大ダメージ。だから最終チェックに徹底的に気をつける。**（職人気質）  
- 絶対にやらない / 譲れないルール：  
  → 特になし

## SECTION 4. ターゲット（誰のためのLPか）

ポートフォリオ型でも「誰向けか」が滲まないLPは記憶に残らない。

- **理想の依頼者像**：  
  → **「自分（依頼者）ではできないことを、代わりにやってくれる人を求めている人」**。  
  → 「人」というより**案件内容で楽しい/楽しくないが決まりがち**（＝人物像で固定するより、案件タイプで方向づけられる職人型）。

- 過去で「この人の仕事は楽しかった」共通点：  
  → 未深掘り（次回ヒアリング候補）

- 来てほしくない依頼：  
  → **販促物・POPで「面白くない案件内容」**。  
  → やれるしやるけど、量産的・指示丸投げ型のものはやりたくないニュアンス。  
  → ⚠️ ここは差別化軸として効くので、**次回もう一段掘りたい**（どんなタイプの販促物が「面白くない」のか具体例で）

> 🔖 **しまの/クロ メモ**：  
> 「人で選ばない、案件で選ぶ」と言える人はかなり少ない。これは「気の合う社長を募集」型LPではなく、**「こういう案件を持ってきてくれる人募集」型LP**にすべき、という強いシグナル。  
> ターゲット像セクションは「ペルソナの肖像」ではなく**「こういう仕事が好きです」リスト**で組む案を持っておく。

## SECTION 5. ストーリー（人柄を伝える素材）

ポートフォリオLPは「作品」だけでなく「人」で選ばれる。

- デザイナーになった理由 / きっかけ：  
  → **高校2年生のとき、テレビドラマでデザイナーが出ているのを見て「私は将来この職業になる」と決めた**。そのまま専門学校へ進学。  
  → 💡 **一貫性の物語**：高2から今まで職業選択がブレていない＝信頼の土台になるエピソード

- 転機・印象的な出来事：  
  → **渋谷の事務所時代**。事務所に5日泊まり込みも普通の環境。その中で **「上司に下剋上してやる」と思ったのが転機**。  
  → 💡 **裏面の強さ**：穏やかな佇まいと、ハングリーで負けず嫌いな根性が同居。

- 届けたい価値・世界観：  
  → **「常に誰かの思い出になったり、ちゃんと残るものを作る」**  
  → **「どんな金額であれお金を出してもらっている以上、精一杯応えたい。出してもらった以上、それに応えられるものでありたい」**  

- **本人発キャッチコピー**：  
  → **「デザインに愛を」** ❤️ ← LPのコア・ステートメント候補

### 🎣🌊🚀🌍 ビジュアル・ストーリー設計コンセプト（しまちゃん発・2026-05-17）

**参考サイト（ヤマト）の演出構造を踏まえつつ、SAYAKA独自のストーリーアニメーションを設計する。**

#### コンセプト原文（しまちゃん発）
> 参考ではトラックが画面下に進んでいくが、釣り竿から垂れるルアーが海の中にどんどん下に降りていくにつれて、深海だとグラデーション的に色が濃くなっていく。本来ならそこにたどり着くんだけど、そこにはたどり着かず、そのまま宇宙に行ってほしい。宇宙に行って地上に帰ってきてっていうデザイン。

#### スクロール連動シナリオ（段階）

| 段階 | 場面 | 背景色 | ビジュアル要素 |
| :---: | :--- | :--- | :--- |
| ① | **海上**（釣り竿） | 白〜淡い水色 | 釣り竿、空、海面 |
| ② | **海中（浅瀬→中層）** | 水色 → 青 | ルアーが沈んでいく、魚、光のゆらぎ |
| ③ | **深海（手前）** | 濃紺 → 黒に近い藍 | 暗くなる、深海生物のシルエット |
| ④ | **突破点**（深海の底に着地しない） | 黒（一瞬の闇） | ルアーが「貫通」する瞬間 |
| ⑤ | **宇宙** | 黒地に星 | ルアー（または変容したもの）が宇宙空間へ |
| ⑥ | **帰還**（地上） | 朝焼け／白基調へ戻る | 地上に降り立つ／届く |

#### モノクロ式・意味の翻訳（SAYAKA価値構造との接続）

このストーリーは偶然ではなく、**SAYAKAの価値構造そのもの**をビジュアル化している：

| ストーリー要素 | SAYAKAの価値との接続 |
| :--- | :--- |
| 釣り竿から垂らすルアー | 顧客から預かる「**依頼**」「**期待**」「**お金**」 |
| 海に潜っていく（深さを増す） | 案件を**深く理解する**プロセス（コミュ力 × ヒアリング） |
| 深海の暗さ（普通ならここが終着） | 「ありきたりな成果物」「普通のデザイン」の到達点 |
| **そこに辿り着かず、突き抜けて宇宙へ** | 「**金額以上に応える**」「**期待を超える**」=下剋上の根性 |
| 宇宙から地上に帰ってくる | 「**思い出に残る／ちゃんと残るもの**」として顧客の手元に届く |

→ つまりこのアニメーションは「**SAYAKAに依頼すると何が起きるか**」を1本のストーリーで体験させる仕掛け。  
→ **キャッチコピー「デザインに愛を」と完全に対**になる演出。  
→ ポートフォリオ型LP（信頼担保）の「信頼の根拠」を、作品列挙ではなく**体験**で見せる構造になる。これはモノクロ式として極めて高品質。

#### 実装上の検討事項（制作フェーズで詰める）

- **技術選定**：スクロール連動アニメーション。GSAP + ScrollTrigger / Lottie / Three.js などが候補
- **パフォーマンス**：背景フルカラー遷移＋オブジェクト追従はモバイルで重くなりがち。**簡略版／フォールバック**を用意
- **テキストの乗せ方**：各段階の脇に SAYAKA の価値ステートメント（「深く聴きます」「期待を超えます」「ちゃんと残します」など）を配置するか、純粋にビジュアルで語るか
- **余白との両立**：演出が凝りすぎると SAYAKAの「シンプル・余白」軸とぶつかる。**色は段階的・要素は最小限**で構成
- **CTA配置**：ストーリー終盤（地上に戻ってきた直後）に問い合わせフォームを置くと、感情的余韻と接続して強い導線になる
- **実装担当の判断**：難度が高いため、リューク or 外部協力者のアサインを検討

## SECTION 8. LPゴール設計（CTA = 訪問者に取らせたい行動）

ポートフォリオ型でも、見終わった人に「次の一歩」を必ず設計する。

- LP訪問者に**最終的に取ってほしい行動**：  
  - [x] **① 問い合わせフォームからメール連絡（メインCTA）**  
  - [x] **② 公式LINE登録（サブCTA / おまけ枠）**  
  - [ ] Instagram フォロー  
  - [ ] メールで直接連絡  
  - [ ] 制作物PDF（ポートフォリオ）DL  

- これがゼロだったらLP失敗：  
  → **問い合わせフォームからのメール連絡（メインCTA）**

- 反応の成功ライン：  
  → 未確認（次回 or 公開前に擦り合わせ）

> 🔖 **しまの/クロ メモ**：  
> メインCTA＝問い合わせフォーム、サブCTA＝LINE という設計が確定。  
> → LPの構成上、フォームは **ファーストビュー（H1直下）／中間（実績後）／フッター直前** の3カ所に配置するのが王道。  
> → LINEは「もう少し緩く繋がりたい人向け」のクッションとして、フォーム横またはセクション間に控えめ配置。
