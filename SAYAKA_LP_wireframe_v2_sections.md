## 1. 00. ヘッダー

**意図**：シンプル軸を最初に印象づける。スクロール中もずっと見えるが、邪魔にならない存在感。

**配置**
- 画面上端に固定（sticky）
- スクロール時は背景半透明白（rgba(255,255,255,0.85)）+ blur
- 高さ：60〜72px

**レイアウト案**
```
SAYAKA              About  Works  Voice  Contact
```
- 左：屋号「SAYAKA」（**テキスト仮置き、後日ロゴ画像に差し替え可能な設計**）
- 右：ナビメニュー4項目（アンカーリンクで各セクションへ）
- スマホ：ハンバーガーメニュー

**ロゴの実装方針（2026-05-19確定）**
- 現時点では正式ロゴなし → テキスト「SAYAKA」で仮表示
- 将来ロゴ画像ができた時のため、HTML/CSSで以下の構造で実装：
  ```html
  <a class="logo" href="/">
    <img src="" alt="SAYAKA" class="logo-img" hidden>
    <span class="logo-text">SAYAKA</span>
  </a>
  ```
- ロゴ画像のパスを指定すると、テキストが自動で消えて画像が表示される（JS/CSSの簡単な制御）
- 場所だけ確保しておく仕組み

**メニュー項目（v2構造に合わせて見直し）**
- About → ヒーロー〜スクロール演出（自己紹介体験全体）
- Works → ワークセクション（10）
- Voice → スクロール演出のStage 5（青空、お客さまの声）。または独立セクション化も検討
- Contact → お問い合わせフォーム（11）

**[要確認]**
- Voiceがスクロール演出の途中だと飛びにくい → 別途独立セクションを作るか、アンカーで該当Stageに飛ばすか

## 2. 01. ヒーロー

**意図**：3秒で「この人はシンプル・余白を大事にする人だ」と伝わる。コアコピー「デザインに愛を。」を最大化。**ファーストインプレッションの100%を担う。**

**配置**
- ファーストビュー（画面いっぱい）
- 中央に大きくH1
- 視線誘導：下にスクロールしてもらうためのサイン

**ビジュアル**
- 背景：白〜うすい水色のグラデ（次のStage 0前奏とつながる色味）
- メインビジュアルはなし。**余白で語る**
- 中央やや上にH1、その下に控えめなサブコピー

**コピー初稿**
- H1：「デザインに愛を。」（既決定、句点あり）
- サブ：「思い出に残るものを、つくります。」
- スクロール誘導：「↓ Scroll」または「下にスクロールしてください」

**アニメ挙動**
- ページロード時：H1が下からふわっとフェードイン（duration 1.0s, ease: power3.out）
- サブコピー：H1の0.4秒後に同じ動きでフェードイン
- スクロール誘導の矢印：ロード完了1秒後に出現、ゆっくり上下に揺れる

**スクロール量**：100vh（ファーストビュー専有）

**[要確認]**
- サブコピー「思い出に残るものを、つくります。」のニュアンスでOKか
- ヒーローに何らかのビジュアル要素（釣り竿のシルエットなど）を入れるか、完全に余白で勝負か

## 3. 02. はじめまして 【省略確定 2026-05-19】

**判断結果**：このセクションは **省略** に確定。

**理由**
- スクロール演出のStage 0前奏「ちょっと、深いところまで潜ってみませんか。」がすでに導入役を担っている
- 02を入れると似た雰囲気が二度続いて間延びするリスクを回避
- テンポの良さを優先

**新しい導入の流れ**
- 01. ヒーロー（静、デザインに愛を。）
- → 03. Stage 0 前奏（動、釣り竿でルアー投げる）

セクション番号としても 02 は欠番扱いとし、本セクション以下の番号は繰り上がらない（Stage 0は引き続き「03」扱い）。

## 4. 10. ワーク（代表作・画像メイン）

**意図**：スクロール演出を通り抜けた後の、**画像で語る代表作展示**。SAYAKAさんの「ベストオブベスト」を見せる。

**配置**
- スクロール演出（Stage 6 地上着水）の直後
- ページの中段〜下段
- 高さ：コンテンツ量に応じて可変（複数viewport）

**ビジュアル**
- 背景：白（#FFFFFF）
- レイアウト：グリッド表示（PC: 3〜4列 / SP: 2列）
- 余白広め、画像メイン
- ジャンル別フィルター（タブ切替）

**構成案**

```
[セクション見出し]
「代表作」（または "Works"）

[ジャンルタブ]
[ すべて | ロゴ | チラシ | パッケージ | グッズ | Web | ... ]

[グリッド]
[画像] [画像] [画像]
[画像] [画像] [画像]
...

[下層誘導]
「もっと見る → フルポートフォリオ」
```

**ジャンルタブ（10ジャンル、Stage 2と同じ並び）**
1. すべて（デフォルト表示）
2. ロゴ
3. 名刺・ショップカード
4. チラシ・フライヤー
5. ポスター
6. パッケージ
7. 販促物・POP
8. グッズ
9. SNSバナー・アイキャッチ
10. パンフレット・冊子
11. Web/LP

**各カードの構成**
- 画像（メイン）
- 案件名 or 「ロゴ・〇〇」のような短いキャプション
- クライアント名（公開OKなもののみ）
- クリックでモーダル拡大（または下層ページに飛ばす）

**選定基準（代表作）**
- 各ジャンルから1〜3点ずつ厳選
- 全体で15〜30点程度
- 棚卸しシート（[[SAYAKA_LP_client_triage.md]]）の「A」ラベル群

**下層ページへの誘導**
- ワークセクションの末尾に「もっと見る → フルポートフォリオ」リンク
- 飛び先：`/works.html`（下層ページ）

**アニメ挙動**
- 各カードがスクロールイン時にフェードイン（stagger 0.1s）
- ホバー時：軽く浮き上がる（translateY -4px）
- ジャンル切替時：フィルター適用でクロスフェード

**[要確認]**
- 代表作の選定（棚卸し完了後に確定）
- モーダル拡大時の挙動（画像のみ拡大 / キャプションと一緒 / 案件詳細ページに飛ぶ）
- カードに「View Details」ボタンを付けるか、画像クリックで直接モーダルか

## 5. 11. お問い合わせフォーム

**意図**：LPの最終ゴール。「相談したい」と思った人が迷わず連絡できる場。

**配置**
- ワーク（10）の下
- ページの下段

**ビジュアル**
- 背景：うすい水色（#F0F8FB）または白
- 中央寄せ、最大幅 720px程度
- フォーム要素は十分な余白で、書きやすさ優先

**フォーム項目**

| 項目 | 必須 | タイプ |
|---|:-:|---|
| お名前 | 必須 | テキスト |
| 会社名・屋号 | 任意 | テキスト |
| メールアドレス | 必須 | email |
| 電話番号 | 任意 | tel |
| 制作物の種類 | 任意 | チェックボックス（複数選択可）：ロゴ / 名刺 / チラシ / ポスター / パッケージ / 販促物 / グッズ / SNS / パンフレット / Web / その他 |
| ご予算（目安） | 任意 | セレクト：〜5万 / 5〜20万 / 20〜50万 / 50万〜 / 相談 |
| 希望納期 | 任意 | セレクト：1週間以内 / 2〜4週間 / 1〜3ヶ月 / 3ヶ月以上 / 相談 |
| ご相談内容 | 必須 | テキストエリア |
| プライバシーポリシー同意 | 必須 | チェックボックス |

**コピー初稿**
- セクション見出し：「お問い合わせ」
- リード：「お仕事のご相談・お見積もり、お気軽にどうぞ。通常2〜3営業日以内にお返事します。」
- 送信ボタン：「送信する」
- 送信完了メッセージ：「お問い合わせありがとうございます。内容を確認のうえ、2〜3営業日以内にご連絡いたします。」

**サブCTA（LINE）**
- フォームの横または下に「もう少し気軽に話したい方は」「LINEからどうぞ」とLINEリンク
- LINEアイコン + 「LINEで相談する」

**実装方針（2026-05-19 一部確定）**
- 送信先メールアドレス：**sycw.1988@gmail.com**（LP上には絶対に表示しない、フォームのバックエンドにだけ設定）
- 送信先サービス：Netlify Forms 推奨（ホスティングをNetlifyにする予定なら統合で楽、いたずら対策にも強い）
- スパム対策：Netlify Forms標準のHoneypot + reCAPTCHA v3（必要なら）
- 送信成功/失敗のフィードバック表示
- メールアドレス漏洩対策：HTMLにmailto: リンクを書かない、JSコード内にもプレーンテキストで埋め込まない

**[要確認]**
- LINE 公式アカウントのURL（準備待ち）
- ご予算と希望納期の選択肢を残すか省くか（負担になりすぎないか）
- プライバシーポリシーの本文（別ページ or モーダル）

## 6. 12. フッター

**意図**：必要最小限の情報と、SNSへの動線。

**配置**
- ページ最下部
- 高さ：120〜180px程度

**ビジュアル**
- 背景：濃いめのグレー（#222222）または白
- テキスト：白 or 黒（背景に応じて）
- フォントは小さめ

**構成（2026-05-19 SNS構成確定）**
```
[SAYAKA ロゴ or テキスト]

[ クイックリンク ]
About | Works | Voice | Contact | プライバシーポリシー

[ SNS ]
Instagram | LINE | Pinterest

[ コピーライト ]
© SAYAKA 2026
```

**SNS構成（確定）**
- Instagram：釣りアカウント（趣味だがLPテーマと整合）
- LINE：公式アカウント（準備中、できたらURL反映）
- Pinterest：ある。グラフィックデザイナーとしてのセンス・世界観を見せる武器として有効
- X（Twitter）：なし、表示しない

**コピー初稿**
- セクション見出し：なし（フッターなので）
- コピーライト：「© SAYAKA 2026. All rights reserved.」

**[要確認]**
- Instagram URL（釣りアカウント）
- LINE 公式アカウント URL（準備待ち）
- Pinterest URL
- フッターの色（白 / 濃いめのグレー / その他）
