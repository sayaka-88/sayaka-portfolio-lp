## 1. サイトマップ（全体構造）

PCで上から下へ、SPでもそのまま縦に積みます。全部で **9セクション**。

```
┌─────────────────────────────────────┐
│ 00. ヘッダー（メニュー）              │
├─────────────────────────────────────┤
│ 01. ヒーロー（H1：デザインに愛を。）  │ ← メインCTA①
├─────────────────────────────────────┤
│ 02. はじめまして（自己紹介の入口）    │
├─────────────────────────────────────┤
│ 03. つくれるもの（サービス／領域）    │
├─────────────────────────────────────┤
│ 04. ストーリー演出（スクロール体験）  │ ← B案・背景色段階遷移
├─────────────────────────────────────┤
│ 05. ワーク（代表作）                  │ ← メインCTA②
├─────────────────────────────────────┤
│ 06. お客さまの声                      │
├─────────────────────────────────────┤
│ 07. わたしのこと（経歴タイムライン）  │
├─────────────────────────────────────┤
│ 08. お問い合わせ                      │ ← メインCTA③ + サブCTA（LINE）
├─────────────────────────────────────┤
│ 09. フッター                          │
└─────────────────────────────────────┘
```

- **メインCTA（問い合わせフォーム）配置**：①ヒーロー直下 ／ ②ワーク後 ／ ③下部のフォーム本体
- **サブCTA（LINE）配置**：08のフォーム横に「もう少し気軽に話したい方はLINEからどうぞ」と添える

## 2. セクション別：意図 × ワイヤー × コピー初稿

### 00. ヘッダー

**意図**：必要最小限。シンプル軸を最初に印象づける。

```
┌────────────────────────────────────────┐
│  SAYAKA          About  Work  Voice  Contact │
└────────────────────────────────────────┘
```

- ロゴ位置：左。テキストロゴで OK（既存ロゴがあれば差し替え）
- メニュー：右。アンカーリンクで各セクションへスクロール
- スクロール時は固定（sticky）／背景は半透明白

### 03. つくれるもの（サービス／領域）

**意図**：何を頼めるかを明示しつつ、「シンプル / 余白 / 装飾過多はやらない」美意識を宣言する。

```
┌────────────────────────────────────────┐
│  つくれるもの                            │
│                                        │
│  ロゴ　名刺　ショップカード              │
│  チラシ　フライヤー　ポスター            │
│  パンフレット　パッケージ                │
│  販促物・POP　SNSバナー                  │
│  看板・サイン                            │
│                                        │
│  ────────────────────────              │
│                                        │
│  得意なのは、                            │
│  シンプルなデザイン、                    │
│  余白がきれいにとれるデザイン。          │
│  ポップな表現も、おまかせください。      │
└────────────────────────────────────────┘
```

**コピー初稿：**
- 見出し：**つくれるもの**
- リスト（横並び or グリッド）：ロゴ／名刺／ショップカード／チラシ・フライヤー／ポスター／パンフレット／パッケージ／販促物・POP／SNSバナー・アイキャッチ／看板・サイン
- 締めの一文：**得意なのは、シンプルなデザイン、余白がきれいにとれるデザイン。ポップな表現も、おまかせください。**

**注記**：

- リスト部分は装飾なしの素朴な文字組み（=世界観を裏切らない）

### 04. ストーリー演出（スクロール体験）

**意図**：ヒアリングシートのコンセプト「依頼 → 深く聴く → 期待を超える → 残る」を、背景色の段階遷移＋短いコピーで体験させる。B案（簡略版）。

**スクロール構成（5段階）：**

```
段階  背景色         コピー（1行）
─────────────────────────────────────────
①    白             あなたの依頼を、お預かりします。
②    淡い水色       ことばの奥にある景色まで、聴かせてください。
③    濃紺           ふつうは、やめておきます。
④    黒に星         期待を、ひとつ超えていきます。
⑤    朝焼け→白     ちゃんと、残るかたちで。
```

**レイアウト（各段階共通）：**
```
┌────────────────────────────────────────┐
│   （背景色：段階ごとに切替）              │
│                                        │
│                                        │
│        ことばの奥にある景色まで、         │
│        聴かせてください。                │
│                                        │
│                                        │
└────────────────────────────────────────┘
```
- 各段階：縦100vh（画面いっぱい）／コピーは中央
- 文字色：背景に応じて自動切替（黒地は白文字、白地は黒文字）
- 段階間：CSSの `transition` でゆっくりフェード
- アイコン等：原則なし。最終形で必要なら最小限のSVGを追加検討

**技術メモ（実装フェーズ用）：**
- `position: sticky` の親コンテナ + IntersectionObserver で背景クラスを切替
- スマホでは段階を3つに圧縮するフォールバックを検討
