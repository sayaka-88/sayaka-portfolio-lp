# SAYAKA LP ワイヤーフレーム v2：スクロール演出パート

## 0. 前提

このドキュメントは、LPのメインボディとなる「**釣り竿でルアーを投げる → 海面 → 海中 → 深海 → 宇宙 → 青空 → 地上着水**」の循環スクロール演出を、各段階ごとに詳細化したものです。

- 演出は **フル版**（GSAP + ScrollTrigger 想定）
- 海中では画像を見せず、ジャンル名のみテキストでチラ見せ
- 画像メインの代表作は、演出が終わった後の独立ワークセクションで展開
- トンマナ「楽しい・ワクワク・暗くしたくない」を全段階に適用

---

## 1. 演出の全体像

```
[ヒーロー：デザインに愛を。]
        │
        ▼
─────────────────────────
 Stage 0  釣り竿でルアー投げる（前奏）
 Stage 1  海面     ：自己紹介
 Stage 2  海中     ：できること（ジャンル名のみ）
 Stage 3  深海     ：キャリア・実績の重み
 Stage 4  宇宙     ：取引先一覧（広がり）
 Stage 5  青空     ：お客さまの声・仕事観
 Stage 6  地上着水 ：ありがとう・CTA予告
─────────────────────────
        │
        ▼
[ワーク：ジャンル別代表作（画像メイン）]
[お問い合わせフォーム]
[フッター]
```

各段階の長さ目安（viewport 比）:

| Stage | 長さ | コピー量 | アニメの重み |
|---|:-:|:-:|:-:|
| 0. 前奏 | 0.7 | 短 | 軽（導入） |
| 1. 海面 | 1.0 | 中 | 中（着水とテキスト出現） |
| 2. 海中 | 2.0 | 多（ジャンル10個） | 中（魚と泡） |
| 3. 深海 | 1.5 | 中 | 中（光と静寂） |
| 4. 宇宙 | 1.8 | 少（読ませない・雰囲気のみ） | 重（塊魂風） |
| 5. 青空 | 1.5 | 中 | 中（雲と日の出） |
| 6. 着水 | 1.0 | 短 | 軽（戻る） |

合計：約 9.5 viewport ぶんのスクロール量。

## 2. 各段階の詳細

### Stage 0：釣り竿でルアー投げる（前奏）

**意図**：ヒーロー直後、ストーリーの世界に引き込む合図。「これから何かが始まる」というワクワク。

**視覚**
- 背景：白〜うすい水色のグラデーション（空）
- 要素：画面右上から伸びる釣り竿のシルエット（細い線）。ルアーが画面中央上方で「ピョン」と弾む
- 太陽の光がうっすら差している（左上から斜めに）

**ポロポロ出すコピー（H2 相当）**
- メイン：「ちょっと、深いところまで潜ってみませんか。」
- サブ：「（スクロールしてください）」※小さく、控えめに

**アニメ挙動**
- 釣り竿はスクロール開始と同時に画面に現れる（フェードイン）
- ルアーは2秒ほど弾んで、スクロール量に応じて画面下方向に落ち始める
- スクロール量の終わり（70%地点）で、ルアーが画面中央付近に到達

**スクロール量**：viewport 0.7 個分

---

### Stage 1：海面（自己紹介）

**意図**：「はじめまして」の最初の挨拶。誰なのかを名乗る。

**視覚**
- 背景：水色（#A8D8E8 〜 #6BB6D8 のグラデ、上が明るく下が濃く）
- 要素：水平線、波紋、光の屈折表現
- ルアーが水面に着水する瞬間→波紋がパッと広がる

**ポロポロ出すコピー**
- 「はじめまして、SAYAKAです。」（H2、大きめ）
- 「グラフィックデザイナー / ディレクター」（サブ、小さめ）
- 「東京を拠点に、デザインの仕事をしています。」（本文）

**アニメ挙動**
- ルアーが画面中央〜やや左で水面に着水（小さなしぶき + 波紋3重）
- 波紋の広がりに合わせて、コピーが順番にフェードイン + Y軸20px上昇
  - 1行目：着水と同時
  - 2行目：0.3秒遅れ
  - 3行目：0.6秒遅れ

**スクロール量**：viewport 1.0 個分

---

### Stage 2：海中（できること = ジャンル名）

**意図**：「何ができる人か」を遊び心ある形で見せる。**画像は一切出さない。** 文字とイラスト（魚）だけ。

**視覚**
- 背景：水色 → 青 → 濃い青へ徐々に深くなる
- 要素：魚たち（シルエットや簡単なイラスト）が左右から横切る。泡が上に流れる。
- 光が上から斜めに差し込む（水中の光のスジ）

**ポロポロ出すコピー（ジャンル名のみ・10個）**

| # | ジャンル名 |
|:-:|---|
| 1 | ロゴ |
| 2 | 名刺・ショップカード |
| 3 | チラシ・フライヤー |
| 4 | ポスター |
| 5 | パッケージ |
| 6 | 販促物・POP |
| 7 | グッズ |
| 8 | SNSバナー・アイキャッチ |
| 9 | パンフレット・冊子 |
| 10 | Web/LP（AI制作） |

※ 看板/サインは「販促物・POP」に統合（独立表記なし）。
※ アーティストグッズ（NFX, JSB, 各種ステッカー等）を考慮し「グッズ」を独立カテゴリに。
※ Web/LP は AI 制作対応として明示。今後の主要な差別化カード。

**アニメ挙動**
- 各ジャンル名は、魚と一緒に左右から泳いで現れる
  - 例：「ロゴ」というテキストが、魚のイラストと並走して画面右から左へ
  - 画面中央で一瞬止まり（読める間）、左へ抜けていく
- 同時に2〜3個のジャンル名が泳いでいる状態が理想（賑やかさ）
- スクロール速度に合わせて魚の泳ぐスピードも変わる（scrub連動）

**スクロール量**：viewport 2.0 個分（10ジャンルあるので余裕を持って）

---

### Stage 3：深海（キャリア・実績の重み）

**意図**：「深く潜る = 経験の深さ」を視覚で表現。トンマナ的に **暗黒ではなく「光が差す深い藍」**。

**視覚**
- 背景：濃い藍（#1A3A5C 〜 #0F2540）
- 要素：中央に1本のスポットライト（上から光が落ちてくる）、または深海魚の灯り
- 周囲は暗いが、ライトの周りだけがやさしく照らされている

**ポロポロ出すコピー**
- 「15年、デザインと向き合ってきました。」
- 「専門学校で、講師もしています。」
- 「商工会のエキスパートバンクに登録されています。」

※ 渋谷時代の「下剋上」エピソード等は不掲載（決定済み）。**事実ベースの誠実な3行**でまとめる。

**アニメ挙動**
- スポットライトの中に、コピーが1行ずつ静かに浮かび上がる
- 3行が出揃うと、ライトがゆっくり明滅する（呼吸のように）
- 文字色は白〜ゴールド（明るめのゴールド）

**スクロール量**：viewport 1.5 個分

---

### Stage 4：宇宙（広がり = 雰囲気だけ、読ませない）

**意図**：深海から一気に視野を広げる。**広がりと冒険のワクワク感**を見せる場。社名は読ませず、雰囲気だけ伝える。詳細な取引先一覧は下層ページに分離。

**視覚**
- 背景：深い藍 → 黒に近い濃紺（#0A1428）グラデ
- トンマナ：**塊魂風**（カラフル・賑やか・ぐるぐる回るユーモラスな宇宙）
  - カラフルな星（ピンク・水色・黄色・白）
  - ふわふわ浮かぶ謎の要素（小さい星形・丸・四角・三角など、塊魂のあのワチャワチャ感）
  - 中央にルアー（または小さな球体）があって、その周りを星や要素がぐるぐる回る
  - 流れ星が画面を斜めに走る
- 注意：トンマナ「上品さ」とのバランスで、塊魂より少しシックなパレットに調整可

**ポロポロ出すコピー（読ませる文字は最小限）**
- ヘッドライン：「これまで、こんな方々と。」（中央付近、大きめ）
- 背景に浮かぶ社名（**読めなくてOK・雰囲気の一部**）：取引先名がふわふわ流れている。意図的に小さい・薄い・ぼやけて配置。「たくさんの方々と関わってきた感」を視覚的に伝える
- 誘導テキスト（下部、控えめに）：「詳しい取引先・実績はこちら →」（下層ページへのリンク）

**アニメ挙動**
- 中央の球（ルアー？）の周りを、星や要素がぐるぐる回り続ける
- 流れ星がランダムなタイミングで斜めに走る
- 背景の社名群はゆっくりと方向を変えながら流れる（読ませる動きではなく雰囲気）
- スクロールが終盤に近づくと、ヘッドラインと誘導テキストがフェードイン

**スクロール量**：viewport 1.8 個分（情報パートが分離したので少し短く、でも演出の派手さは保つ）

**[役割分担]**
- このStage 4 = 「広がりの雰囲気」だけ伝える演出
- 詳細な取引先一覧・実績画像・パスワード保護案件は **下層ページ `/portfolio.html`** に分離
- 下層ページの構成は [[SAYAKA_LP_subpage_v2.md]] で別途仕様化

### Stage 5：青空（お客さまの声・仕事観）

**意図**：宇宙の広がりから、日常に戻ってくる温かさ。**人として知ってもらう**場。

**視覚**
- 背景：朝焼けのグラデ（下からオレンジ → ピンク → 水色 → 上は青空）
- 要素：雲がゆっくり流れる、太陽が地平線から昇る
- 全体的に明るく、希望感のあるトーン

**ポロポロ出すコピー**
- ヘッドライン：「よくいただく、こんな言葉。」
- 引用1：「他とは違うっていう感想がもらえて、嬉しかった。」（仮）
- 引用2：「金額以上の仕上がりで、お得な気がした。」（仮）
- 引用3：「思い出に残るもの、つくれた気がする。」（仮）
- 締め：「思い出に残るものを、つくります。」（仕事観として）

※ 引用は「よくいただく言葉」型で確定済み（実名引用ではない）。文言は要相談。

**アニメ挙動**
- 太陽が昇るのに合わせて、雲が左右から流れてくる
- 雲の上に引用テキストが乗ってきて、左から右へゆっくり流れる
- 締めの一文だけ、画面中央で静止して印象づける

**スクロール量**：viewport 1.5 個分

---

### Stage 6：地上着水（ありがとう・CTA予告）

**意図**：ストーリーの締めくくり。**戻ってきた安心感**と、次のアクションへの自然な誘導。

**視覚**
- 背景：青空 → 白っぽい朝の光
- 要素：地面（緑のラインか、桟橋のシルエット）、ルアーが釣り竿に戻ってくる or 静かに着地
- 全体的に静かで、満たされた雰囲気

**ポロポロ出すコピー**
- 「ここまで読んでくださって、ありがとうございます。」
- 「ご相談・お見積もり、お気軽にどうぞ。」
- CTA予告：「↓ お問い合わせフォーム」 or 「↓ 代表作を見る」（次セクション誘導）

**アニメ挙動**
- ルアーが画面上から戻ってきて、釣り竿のラインに収まる
- 同時に、画面下に向かって自然に視線が流れるよう、コピーが下から上に積み上がる
- 最後にCTA予告がふわっと現れる

**スクロール量**：viewport 1.0 個分

## 3. 共通ルール（ポロポロ演出）

### テキストの出現
- フェードイン + Y軸20px の上昇
- duration: 0.6秒
- ease: power2.out
- stagger: 0.3秒（複数行ある場合）

### スクロール連動
- ScrollTrigger の scrub: 1（スクロール量に対してアニメ位置が1:1で連動）
- pin: 各stageは原則pinで止める（背景が固定された状態でテキスト・要素が動く）

### 背景色のトランジション
- 各stage間で背景色がグラデーション的に切り替わる
- 切り替えは scrub 連動でスムーズに

### 縦持ち（スマホ）対応
- スクロール量は維持（モバイルでは1 viewport が小さいので、より長く感じる可能性あり）
- 魚や星の位置は、画面幅に応じて自動調整
- フォントサイズはレスポンシブ（H2 で PC 48px → SP 32px 程度）
